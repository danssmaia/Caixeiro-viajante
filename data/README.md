# DADOS

Os dados das regiões da Bahia estão nessa pasta.

```
.
├── data
│   ├── coords.csv        # Arquivo CSV contendo as coordenadas das cidades da Bahia
│   ├── coordsFront.csv   # Arquivo CSV contendo os indices, coordenadas e indices que fazem fronteira
│   ├── custos.csv        # Arquivo CSV contendo o custo de deslocamento entre as cidades
│   ├── mapapv.png        # Mapa do estado da Bahia em escala de cinza, com pontos vermelhos nos números da regiões. Utilizado para extrair a posiciao de cada índice.
```
